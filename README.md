# Towny Residency
Towny Residency is a plugin that adds a "Residency Paper" a form of ID Card, intended to use alongside Towny (hard dependency) for RP purposes.

## Commands:
**residency:**\
description: Gives out your own Residency Paper. \
usage: /residency
